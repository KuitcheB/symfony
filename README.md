"# symfony" 
1- télécharger le projet : git clone git@github.com:KuitcheB/symfony.git
2- télécharger les vendors: composer install
3- Modifier l'adressse d'envoi du recapitulatif par mail dans les fixutures pour chaque département
4- charger les fixtures: php bin/console doctrine:fixtures:load

Lancer le serveur d'hebergement symfony avec la commande: php bin/console server:run
